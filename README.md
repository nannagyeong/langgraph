# LangGraph 기반 멀티 에이전트 운동 추천 시스템

사용자의 운동 관련 질문을 입력받아, 질문 속 현재 상태를 추출하고 그에 맞는 운동 후보를 추천한 뒤 최종적으로 정리된 답변을 생성하는 프로젝트입니다.  
LangGraph를 이용해 **질문 분석 → 운동 추천 → 답변 생성** 흐름을 단계별로 분리했고, Ollama의 로컬 LLM을 연결해 멀티 에이전트 구조를 구현했습니다.

---

## 멀티에이전트 소개

보통은 하나의 LLM에게 질문을 바로 넣고 답변을 생성하게 만들 수 있습니다.  
하지만 운동 추천처럼 먼저 사용자의 상태를 파악해야 하는 질문은, 한 번에 처리하면 상태 해석과 추천 결과가 섞이면서 출력이 불안정해질 수 있습니다.

예를 들어 아래와 같은 질문이 들어왔을 때,

> 체력이 안좋고, 살이 계속 찌는데 어떤 운동을 할까?

질문 전체를 한 번에 처리하면 현재 상태를 명확히 정리하지 못하거나, 사용자 상황과 맞지 않는 운동이 추천될 수 있습니다.  
그래서 이 프로젝트에서는 질문 처리 과정을 3단계로 나눴습니다.

- **Extractor Agent**: 질문에서 현재 상태 추출
- **Matcher Agent**: 상태를 바탕으로 운동 후보 추천
- **Answer Agent**: 상태와 운동 후보를 바탕으로 최종 응답 생성

이렇게 역할을 나누면 각 단계의 출력이 더 명확해지고, 결과가 이상할 때 어느 부분을 수정해야 하는지 확인하기도 쉬워집니다.

### 최종 응답

| 분류 | 세부 내용 |
|------|-----------|
| 현재 상태 요약 | 체력이 낮은 상태로 체력 향상이 필요함<br>체중 증가 우려로 꾸준한 운동이 중요함 |
| 추천 운동 | **걷기**: 일상 활동에 쉽게 포함할 수 있는 운동으로, 체력을 천천히 향상시키며 체중 관리에도 도움<br><br>**가벼운 스트레칭**: 몸의 유연성을 높이고 근육을 이완시켜 부상 위험을 줄이며 가볍게 시작할 수 있음<br><br>**요가 동작**: 정신 안정과 함께 근력과 유연성을 동시에 향상시켜 체력과 체중 조절에 긍정적 영향<br><br>**수영**: 저충격 운동으로 근육에 부담이 적고 전신 운동이 가능하여 효율적인 체력 향상과 체중 조절에 효과적<br><br>**자전거 타기**: 지구력과 근력을 키우는 좋은 방법으로, 관절에 덜 부담스럽고 심폐 기능 향상에 도움<br><br>**계단 오르내리기**: 일상생활에서 쉽게 실천 가능하며, 계단 오르내림은 하체 근력 강화에 효과적 |
| 운동 시작 팁 | **점진적 증가**: 체력이 낮다면 처음부터 과도한 운동은 피하고, 현재 수준에 맞는 짧고 쉬운 코스로 시작하세요.<br><br>**일상 통합**: 걷기나 계단 오르내리기와 같이 일상 생활에 쉽게 통합할 수 있는 운동부터 시작해보세요.<br><br>**스트레칭 집중**: 스트레칭은 특히 몸의 유연성과 긴장 완화에 도움이 되므로, 매일 짧게라도 꾸준히 실시하세요.<br><br>**목표 설정**: 작은 목표를 세우고 이를 달성할 때마다 자신에게 보상을 주는 것이 동기 부여에 효과적입니다. |
